| Pergunta | O que valida |
|---|---|
| Pré P4 | % de contadores que nunca falaram — prova o canal dormindo |
| Pré P7 vs Pós P5 | Delta de intenção = impacto real do sistema (H2) |
| Pré P9 vs Pós P4 | Redução de dificuldade percebida (H1) |
| Pós P7 | Se contador = gatilho de confiança para agir |
| Pós P8 | Peso da indicação do contador na conversão (hipótese central) |
| Pós P9 | NPS — KPI de equilíbrio |
| Pós P10 | Mercado para escalar IncentivaBR além da cultura |
